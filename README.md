# House Price Prediction using Machine Learning

## Overview

This project predicts house prices using Machine Learning techniques on the California Housing Dataset. The objective is to build a regression model capable of estimating house prices based on various housing and demographic features.

## Dataset

The project uses the California Housing Dataset available in Scikit-Learn.

### Features

* MedInc (Median Income)
* HouseAge
* AveRooms
* AveBedrms
* Population
* AveOccup
* Latitude
* Longitude

### Target Variable

* Median House Value
